1Building the Teletype Castle

As part of my ongoing quest to increase productivity, I have decided to build a digital cabin in the woods. A hideout of sorts, a retreat from distractions, a castle high on the mountainside. I’m sure Ralph Waldo Emerson would be very proud.

It’s really just a partition on my hard drive: a 32gb ext4 partition with Debian GNU/Linux. It has no X-server, no networking, just emacs.

I created a 16gb fat32 partition as well, to serve as mutually accessible storage space. Whatever programs or stories or work that gets done in castle can be saved onto the bridge partition and accessed by OSX.

The steps I followed to dual boot MacOS and Debian are summarized below:

  1. Install MacOS
  2. Download and unzip rEFInd
  3. Create Linux, swap and storage partitions under OSX
  4. Download .deb packages you want in the castle and save them to the storage partition
  5. Install Debian using the nonfree firmware iso (to get networking to work for the netinstall)
  6. Reboot & Reset NVRAM (Command + Option + P + R)
  7. Using the recovery mode terminal (So SIP is disabled) run the refind-install script
  8. Run dpkg-reconfigure console-setup (switch to a 32 font to accommodate the retina screen)
