How I make the site images

  1. animate the images? dots move and grow?
  2. get whatever image I want to convert into GIMP
  3. Image -> mode -> greyscale
  4. Colors -> Levels -> Set output in range [180,220] to wash the photo out. A higher number will wash the photo out more, resulting in more white and fewer dots in the final image.
  5. Image -> mode -> Indexed. With settings: 1-bit palette / Color dithering: Floyd-Steinberg (normal)
  6. Image -> Mode -> RGB
  7. Paint bucket tool to fill in white bg the offwhite #f6e8e5 of the site.
  8. Save as png into site folder
