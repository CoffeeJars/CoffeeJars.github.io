I wrote a very simple python bot using the praw (Python Reddit API Wrapper) library, so that these unbeatable deals could be better shared with the student body!

#stephen duncanson
#stephen.duncanson@uconn.edu

reddit = praw.Reddit(client_id='',
                     client_secret='',
                     password='',
                     user_agent='',
                     username='')

subreddit = reddit.subreddit('uconn')

while True:
	new_uconn = subreddit.new(limit=1)
	for submission in new_uconn:
		time.sleep(600)
		print(submission.title)
		submission.reply(random.choice(deals))
		print("Replied!")
