How to make halftone images in GIMP

  1. Load original image in GIMP
  2. Image -> mode -> greyscale
  3. Colors -> Levels -> Set output in range [180,220] to wash the photo out. The higher these values the fewer dots in the final image.
  4. Image -> mode -> Indexed. With settings: 1-bit palette / Color dithering: Floyd-Steinberg (normal)
  5. Image -> Mode -> RGB
  6. Color -> Map -> Color Exchange to set bg and fg colors
  7. Image -> mode -> Indexed (1-bit optimal)
