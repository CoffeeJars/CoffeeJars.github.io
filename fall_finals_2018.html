Rewrite 3sin(x) + 6cos(x) in the form Asin(x+`phi`)

All we need to remember for this problem are two identities:

  1. sin(`alpha + beta`) = `cos(alpha)sin(beta) + sin(alpha)cos(beta)`
  2. `cos^2(theta)+sin^2(theta)=1`

By setting the original equation equal to the sum identity, we get an equation that can be simplified:

`3sin(x) + 6cos(x) = Acos(phi)sin(beta) + Asin(phi)cos(beta)`

Looking at the balanced equation, you can see that if it's balanced then:
`Acos(phi) = 3`
`Asin(phi) = 6`

This is because on the left side of the equation sin(x) is being multiplied by 3, on the right sin(`beta`) is being multiplied by `Acos(phi)`. Therefore: `Acos(phi) = 3` and `Asin(phi) = 6`.

Since we know the value of Asin(`phi`) and Acos(`phi`) we can use the second Pythagorean identity to solve for the value of A.

`A(cos^2(phi) + sin^2(phi) = 1)`
`A^2cos^2(phi) + A^2sin^2(phi) = A^2`
`3^2 + 6^2 = A^2`
`A = 3sqrt(5)`

Now we have everything we need to solve for `phi`.

`3sqrt(5)cos(phi) = 3`
`cos(phi) = 3/(3sqrt(5))`
`phi = cos^-1 (1/sqrt(5))`
